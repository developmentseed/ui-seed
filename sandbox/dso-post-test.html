Dropcap

The dropcap is the capitalized first letter of a paragraph. It’s not for every occasion, but when the post/page starts with a paragraph that spans at least two full lines it should be added as it has a big impact on the look and feel of the post.

Unlike this example there should be only one dropcap and only at the beginning of the post.

Content media

Images for content should be placed inside /assets/graphics/content/prose/.
These images are not processed because there isn’t a one size fit them all, so some extra care is needed, specially when it comes to file size:
- Jpg is best suited for photographs and png for graphics.
- Cut to 1500 width (except for full bleed images)
- Compress jpg (70 should be enough). As an alternative you can use compressor.io

Tests will fail if an image is over 500KB If it really needs to be over this size its name can be prefixed with big-- and it will be skipped. Only use when everything else fails.

To ensure correct styling, the content media (image, video, chart…) should be wrapped in a <figure> element with the respective <figcaption> if it exists.

<figure class="align-center">
                    <img src="http://loremflickr.com/1440/360/lego" alt="Image" width="1440" height="360" />
                    <figcaption>Wow, this is a really cool caption!</figcaption>
                  </figure>

It is possible to align media elements by using align-left, align-center, and align-right. Except for centered media, note that text will wrap around it.

It is also possible to use a media element that spans the full width of the page by using the .bleed-full class.

<figure class="bleed-full">
                    <img src="http://loremflickr.com/1440/360/lego" alt="Image" width="1440" height="360" />
                    <figcaption>Wow, this is another cool caption!</figcaption>
                  </figure>

                  {% endhighlight %}

Definition lists

A regular definition list looks like this:

Term B
Definiton/description B

They can also be horizontal:

Social networks
Twitter: @developmentseed
Flicker: developmentseed
Email

Or presented like a table with the use of a class:

Twitter
@developmentseed

info@developmentseed.org
